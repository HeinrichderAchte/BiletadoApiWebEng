# Biletado API - An API for Reservations for WebEngineering 2 Class

 The Project implements the Reservation API to run in an Kind-Cluster via Podman. Therefore "reservation" is a Pod. 

## Setup

You can download the image via the "Package" Option of GitHub.
There are also older versions available but they will therefore not work as intended. 
To Setup you must proide your details in the "appsettings.json".


 - ConnectionString: Here you configure your Database Details
 - Authentication: Cause of time issues this feature is not yet available. It will be availabel in the future. Cause of that it is set to "Enabled" : false
 - Serilog: Here you can provide the details for your logging. You can choose a minimumLevel for Logs and choose to write Logs only to console or also to file
